# Back-end

API inicial para o dominio de loja de imoveis.

## Estrutura

- `src/models`: contratos de dados.
- `src/types`: tipos auxiliares.
- `src/data`: mock inicial.
- `src/controllers`: regras de manipulacao.
- `src/routes`: exposicao de rotas.
- `src/server.ts`: bootstrap do servidor.

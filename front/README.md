# Front-end

Aplicacao React + TypeScript com Bootstrap via CDN e proposta visual classica.

## Estrutura

- `src/routes`: rotas principais com React Router.
- `src/pages`: telas da aplicacao (home, produtos, carrinho, login e cadastro).
- `src/components/layout`: casca visual compartilhada.
- `src/components/home`: hero, dashboard e depoimentos.
- `src/components/produtos`: grid, filtros, card e modal de detalhe.
- `src/components/carrinho`: listagem de itens e resumo.
- `src/components/auth`: formularios de login e cadastro.
- `src/store`: estado global da loja.
- `src/types`: contratos de dados TypeScript.
- `src/styles`: identidade visual e refinamentos.
